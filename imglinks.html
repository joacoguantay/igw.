Uso de etiquetas multimedia y enlaces en HTML

Etiqueta <img>

Se usa para insertar imagenes. Es una etiqueta vacia, no tiene cierre.

>

Etiqueta <video<

Permite insertar videos. Se recomienda usar varios formatos para compatibilidad.

Video de YouTube

Para insertar un feed de YouTube, usa la etiqueta <iframe>.

Feed de Instagram

Para insertar un feed de Instagram, usa la etiqueta <blockquote>.

Etiqueta <a>

Sirve para crear enlaces a otras paginas, documentos o ubicacion.
